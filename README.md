Installation
---

```bash
$ brew update
$ brew install rbenv
$ rbenv install 2.3.8
$ rbenv rehash
$ rbenv global 2.3.8
$ gem install bundler
$ bundle install
```

Run
---

```bash
$ bundler exec jekyll serve
```
