## Mac에서 Homebrew 사용하기 + Sublime Text 사용하기

```shell
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
위의 명령어를 입력해주면 됩니다.

너무 간단합니다.

그리고 터미널에서 
```
brew update
```
만 해주면 잘 설치되었는지 확인할 수 있습니다.
